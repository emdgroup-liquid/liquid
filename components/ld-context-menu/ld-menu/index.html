ld-menu #

The ld-menu component is used internally for the ld-context-menu. It is the visual part of the actual menu without the floating/tooltip functionality.

Examples #

Basic #

<ld-menu>
  <ld-menuitem>Menu item 1</ld-menuitem>
  <ld-menuitem>Menu item 2</ld-menuitem>
  <ld-menuitem>Menu item 3</ld-menuitem>
</ld-menu>

  <li class="ld-menuitem" role="none">
    <button class="ld-menuitem__button ld-button ld-button--justify-start ld-button--ghost ld-button--neutral-ghost" role="menuitem">

  </li>
