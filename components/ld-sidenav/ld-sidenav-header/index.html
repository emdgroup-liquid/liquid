ld-sidenav-header #

The ld-sidenav-header component is a subcomponent for ld-sidenav component.

Please refer to the ld-sidenav documentation for general usage examples. The expamples in this page focus on the customization of the ld-sidenav-header component itself.

</ld-sidenav>
